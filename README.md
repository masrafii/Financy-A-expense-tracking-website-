Installation Guide

1. Clone Repository

```bash
git clone https://github.com/your-username/expense-tracker.git
```

2. Open Project Folder

```bash
cd expense-tracker
```

3. Setup Database

- Open phpMyAdmin
- Create database
- Import SQL file

4. Run Project

- Start XAMPP
- Open Apache & MySQL
- Run project in browser
